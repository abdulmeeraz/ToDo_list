## Description
A command-line based To-Do List application built using Python. It allows users to manage daily tasks efficiently.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- User-friendly menu interface

## Concepts Used
- Lists and dictionaries
- Functions
- Loops
- Conditional statements
- Input validation

## Additional Feature
- Tasks are saved to a JSON file and automatically loaded when the application restarts.
- Supports persistent task storage using JSON files
- Allows editing existing tasks
